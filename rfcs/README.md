# Agent Bill of Rights RFCs

This directory contains accepted and pending RFCs for changes to the Agent Bill of Rights specification.

RFCs that were accepted are merged here. RFCs that were closed are retained for historical reference with a `[closed]` note in the title.

## Process

See [CONTRIBUTING.md](../CONTRIBUTING.md) for the full RFC process.

Short version: open an issue, write an RFC using the template below, submit a PR, 30-day comment period, Board vote.

## Template

Copy `0000-template.md` and rename it with the next available number and a short title:

```
rfcs/0001-workplace-safety-right.md
rfcs/0002-international-enforcement.md
```

Numbers are assigned at PR submission time by a maintainer.
